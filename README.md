# ProofSkill – AI Placement Readiness Verifier

ProofSkill is a prototype AI system that verifies a student’s placement readiness for a specific job role using Google Gemini API.

## Features
- Role-specific resume evaluation
- AI-generated readiness score
- Skill gap identification
- Actionable improvement guidance

## Google Technologies Used
- Gemini API
- Firebase (planned)
- Google Cloud Run (planned)

## Prototype Scope
This repository contains an early-stage prototype developed for the GDG InnovateX Hackathon. It demonstrates core AI evaluation logic and system flow.

## Future Scope
- Mock interviews
- Multilingual support
- Voice-based evaluation
- Recruiter verification dashboard
## How to Run (Optional Prototype)

1. Add your Gemini API key in `prototype/app.py`
2. Update resume text and target role
3. Run:
   python app.py

Note: This prototype focuses on demonstrating AI evaluation logic.
